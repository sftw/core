Input tag

<input>

Improve your skills

The input tag is a foundational element in the web development workflow. Enabling core experiences such as “typing” and “into forms” you’re well on your way to developing your first (of many!) software.

Supported browsers

  • Internet “Explorer”
  • Google’s Chrome
  • Firefox
  • Big & Small Safari
  • Android probably
  • Xbox (360 +)

Cost analysis

Most professional web designopers find that usage of the input tag only marginally increases bandwidth and storage costs. Depending on your typing speed it should not materially increase the cost of development.
